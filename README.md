
👩‍💼 HR Attrition Prediction App

This project is a Machine Learning-based web application designed to predict whether an employee is likely to leave an organization (attrition) based on various personal and professional attributes.

🚀 Key Features
Predict employee attrition using real-world HR data
User-friendly input form for employee details
Supports multiple features like:
Age, Salary, Job Role, Work Experience
Job Satisfaction, Work-Life Balance
Performance Rating, Promotions
Instant prediction output (Attrition: Yes/No)
Helps HR teams make data-driven decisions
